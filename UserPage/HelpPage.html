How to Use This Website

Welcome to the WCSS Tech Booking Site! Follow these steps to book an event:

  1. Navigate to the Book a Tech Event page by clicking the "Book a Tech Event" link in the sidebar.
  2. On the booking page, fill out the required fields:
    • Full Name: Enter your full name.
    • Email Address: Provide a valid email address for communication.
    • Booking Date: Select the date of your event using the date picker.
    • Booking Time: Choose the time of your event using the time picker.
    • Location: Enter the location of the event, e.g. 'Small Gym', 'Room 225', ect.
    • Equipment Needed: List any equipment you require for the event.
    • Additional Notes: Add any extra details or special requests.
  3. Once all fields are completed, click the Submit Booking button to finalize your request.
  4. If you need to return to the main page, click the Back to Main Page button.

If you encounter any issues, please contact the WCSS Tech Crew for assistance.
